Methods

builder(instance, keynullable, isListnullable) → {Promise}

Transform the field on the instance, using another Transformer. This can be used when transforming eargerly loaded relations on an instance ie. Family -> Person or obviously values we would like to transform using a prebuilt Transformer. If the value being transformed is an Instance (Sequelize), we call its get() method to invoke all getters on fields and strip other Sequelize stuff off of the data before transforming it. @see http://docs.sequelizejs.com/class/lib/model.js~Model.html#instance-method-get
Parameters:
Name Type Attributes Description
instance Object the source object.
key String <nullable>
 a key on the source object that can be used to retrieve the field value.
isList boolean <nullable>
 whether the value being transformed should be iterated into the builder.
